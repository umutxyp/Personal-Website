## Find Servers Players Actually Join

MCStat.org is a real-time Minecraft server discovery and analytics platform. Unlike static server lists, every stat — player count, uptime percentage, vote tally — updates live so players always see accurate data before they connect.

- **Live Player Counts:** Pings every tracked server on a short interval and surfaces real-time online counts.
- **Vote Rankings:** Community voting system that surfaces the most active and trusted servers.
- **Uptime History:** Per-server uptime graphs and percentage scores so players can trust a server's reliability.
- **Detailed Server Pages:** Version info, server type tags (SMP, PvP, Prison, etc.), copy-IP button, and player profiles.
- **Leaderboards:** Global rankings by player count, votes, and uptime for competitive visibility.
- **Forum & Skins:** Community forum for server owners and a skin browser integrated into player profiles.
