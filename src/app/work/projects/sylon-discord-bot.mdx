## AI-Powered Server Protection

Sylon is a Discord moderation bot that uses AI to automatically detect spam, ads, and unwanted content — keeping servers clean without manual intervention. Built to scale from small communities to large servers with 20K+ users.

- **AI Anti-Spam:** Machine learning filters that adapt to new spam patterns in real time.
- **Ad Detection:** Automatically identifies and removes promotional content based on context.
- **Server Guard:** Raid protection, alt account detection, and role-gated join flows.
- **Welcome Automation:** Customizable welcome messages, role assignment, and onboarding flows.
- **Activity Logs:** Full audit trail of moderation actions, joins, leaves, and message edits.
- **Ticket System:** Built-in support ticket creation and management directly in Discord.

## Tech Stack

Node.js · Discord.js · AI/ML classification · PostgreSQL · Redis · Docker · Cloudflare Workers.
